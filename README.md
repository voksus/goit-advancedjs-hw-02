# Тема 4. Асинхронний JavaScript і проміси

Цей репозиторій містить виконання домашньої роботи з курсу Advanced JS. Проєкт
включає два незалежних завдання, об'єднаних єдиним візуальним стилем та високими
стандартами юзабіліті.

## 🚀 Що робить цю роботу особливою?

Проєкт не просто виконує базові технічні вимоги (яких було дотримано на 100%), а
й містить додаткові UI/UX покращення. На них було витрачено чимало часу, щоб
результат виглядав повноцінним "живим" продуктом:

- **⌚ Ідеальна синхронізація часу:**  
  Логіка зворотного відліку в таймері (Завдання 1) була серйозно вдосконалена.
  Таймер враховує усунення мілісекундної похибки та працює ідеально синхронно
  відносно системного годинника, усуваючи неприродні "зависання" на останніх
  нульових секундах.
- **✨ Мікроанімації та візуальний фідбек:**  
  Замість сухої зупинки таймера, реалізовано приємний кольоровий блік
  (підсвітку) на цифрах лічильника та полі введення в мить нульової відмітки, що
  дає інтуїтивне розуміння завершення процесу.
- **💬 Кастомізація iziToast:**  
  Сповіщення не виглядають стандартно! Проведена ретельна, прискіплива ручна
  кастомізація кожної спливашки. Налаштовані індивідуальні анімації
  (`bounceInRight`, `fadeOutUp`, `fadeIn`), власні прогрес-бари, точні позиції
  та кастомна поведінка (наприклад, `pauseOnHover: false`).
- **🎯 Таргетована вспливашка "Done!":**  
  При завершенні зворотного відліку з'являється унікальна привітальна
  вспливашка, яка відмальовується не з країв екрану, а **прив'язується
  безпосередньо до цільового елемента (лічильника)** в DOM, і технічно коректно
  видаляє свій якір після закриття.
- **🎨 Преміальний UI/UX:**  
  Кастомні стилі для радіокнопок, акуратні тіні (shadow-box), узгоджені
  відступи, гармонійна типографіка. В стилях навіть додано опціональне
  використання `font-variant-numeric: tabular-nums` у файлі
  `src\css\1-timer.css` на 98-му рядку для позбавлення _стрибання_ цифр таймера
  (актуально, якщо вимкнути використання моноширинного шрифту). Всі сторінки
  мають сучасний, збалансований дизайн.

## 📁 Структура проєкту

- `1-timer.html` + `src/js/1-timer.js` + `src/css/1-timer.css` — Таймер
  зворотного відліку (flatpickr + iziToast).
- `2-snackbar.html` + `src/js/2-snackbar.js` + `src/css/2-snackbar.css` — Форма
  генератора промісів з імітацією затримки (setTimeout).

## 🛠 Стек технологій

- HTML5 / CSS3
- JavaScript (ES6+ модулі, Promises, Інтервали)
- Зовнішні бібліотеки: `flatpickr`, `iziToast`
- Інструмент збірки: `Vite`
